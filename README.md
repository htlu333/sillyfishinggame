# 钓鱼：然后捡到女高中生

一个轻量的单文件钓鱼小游戏：等鱼上钩后用 QTE 抢拉力，卖鱼赚钱，去集市买鱼饵/鱼竿/礼物，再送给漂流瓶事件里的女高中生林汐，推进好感与救援。

## 玩法亮点
- QTE 提竿：鱼上钩后显示按键序列（大鱼更长），在判定时间内按对；鱼竿越好，判定时间越宽松。
- 多场景 + 天气/时间：小溪/河流/湖泊搭配随机天气和时间段，影响上钩速度与稀有度。
- 经济循环：钓到的鱼进入“鱼袋”，在集市一键卖鱼换金币。
- 道具成长：
	- 鱼饵：高级蚯蚓提升稀有鱼率，路亚加强史诗鱼机率。
	- 鱼竿：碳素/竞赛竿延长 QTE 判定窗口。
	- 礼物与烹饪：买奶茶、蛋糕，或用卡式炉把鱼烤熟成“烤鱼”。
- 林汐事件：
	- 河流/湖泊可随机钓起漂流瓶解锁事件入口。
	- 在“林汐的临时营地”送礼/送鱼/送烤鱼提升好感，满足每日小心愿可加成。
	- 补给累计 ≥ 8kg 可安排救援；救出后缩短上钩等待，信任越高加成越大。
	- 若多日未赠送，会轻微掉好感。

## 操作说明
- 开始钓鱼：点击“开始钓鱼”。
- 捕获判定：看到“上钩！”后按提示的按键序列（最后一键总是空格），限定时间内按对即成功。
- 取消钓鱼：点击“取消钓鱼”。
- 场景切换：在家选择“小溪 / 河流 / 湖泊”。
- 集市：在“卖鱼/买鱼饵礼物”按钮进入，可卖鱼、买鱼饵/鱼竿/礼物/卡式炉。
- 事件入口：漂流瓶触发后，家中的“事件地点”会亮起“前往林汐的浅滩”。

## 运行方式
```bash
python ./fishing_game.py
```

依赖：Python 3，自带 tkinter（Windows 默认内置）。

## 存档
- fishing_stats.json 会记录鱼类统计、鱼袋、金币、道具、以及林汐事件进度。

## 小贴士
- 想快点遇到漂流瓶，多去河流或湖泊钓几次。
- 想要大鱼：雨天 + 路亚假饵 + 好鱼竿 能拉长判定并提高史诗鱼率。
- 每日委托偏好：甜食、热食或特定鱼类，尽量对症下药，好感加倍。
